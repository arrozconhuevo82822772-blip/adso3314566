Work flow - Markup & Styling

Anidar Elementos >

Ejemplo:

ul>li>a

Resultado:

<ul> <li><a href=""></a></li> </ul>

Elementos Hermanos +

Ejemplo:

section+header+nav

Resultado:

<section></section> <header></header> <nav></nav>

Repetir Elementos *

Ejemplo:

ul>li*3

Resultado:

<ul> <li></li> <li></li> <li></li> </ul>

Atributo Clase .

Ejemplo:

div.contenedor

Resultado:

<div class="contenedor"></div>

Múltiples Clases .

Ejemplo:

div.clase1.clase2.clase3

Resultado:

<div class="clase1 clase2 clase3"></div>

Atributo Identificador #

Ejemplo:

nav#menu

Resultado:

<nav id="menu"></nav>

Personalizar Atributos [ ]

Ejemplo:

a[href=www.google.com]

Resultado:

<a href="www.google.com"></a>

Números Consecutivos $

Ejemplo:

ul>li.num$*3

section>article.sec$$$*3

Resultado:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul> <section> <article class="sec001"></article> <article class="sec002"></article> <article class="sec003"></article> </section>

Dirección Números Consecutivos @-

Ejemplo:

ul>li.num$@-*3

Resultado:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

Base Números Consecutivos @n

Ejemplo:

ul>li.num$@3*4

Resultado:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

Subir Nivel ^

Ejemplo:

section>article>div^p

Resultado:

<section> <article> <div></div> </article> <p></p> </section>

Agrupar Elementos ( )

Ejemplo:

(header>nav)(section>article)

Resultado:

<header> <nav></nav> </header> <section> <article></article> </section>

Documentos HTML

Ejemplo:

html:xs → XHTML 1.0 Strict

html:xt → XHTML 1.0 Transitional

html:4s → HTML 4.01 Strict

html:4t → HTML 4.01 Transitional

html:5 → HTML5

Resultado (estructura básica):

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Document</title> </head> <body> </body> </html>

Meta Viewport

Ejemplo:

meta:vp

Resultado:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
